# BLACKPILL-Diary
I am continuing my journey in learning bare-metal and some cool embedded stuff. I’m going to update everything related to the Black Pill in this repository, since I am learning using the Black Pill.

The STM32 Black Pill is a great microcontroller development board on a budget. Some people may prefer the Blue Pill because it is cheaper, around $2–$3. However, I was having a lot of issues. I tried tracking variables in MCU Viewer and even in STM Studio, but it didn’t work—I just got flat graphs.

Maybe my board is broken; I’ll test with another board someday. But I suspect my Blue Pill might be a fake one, and the clock configuration could be broken. Unfortunately, I haven’t learned much about clock configuration yet. After learning it, I will definitely try these things again with the Blue Pill as well.
